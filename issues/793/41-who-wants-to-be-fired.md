## 文章摘要

这是一个由 HN 用户自发创建的"另类招聘帖"——题图直白：被你目前的工作搞得想离职/想被裁的人，来这里发泄一下，顺便看能不能得到一些有用的反馈。题主把它定位成"安静辞职/quiet quitting"概念在 AI 时代的延伸版本。

整条帖子读下来，能看到几种典型情绪：

**第一种是"AI 把工作的乐趣抽走了"**。多个用户提到自己在大公司被分到搞"non-deterministic agent flows"——非确定性的 LLM 工作流——再用更多的"AI slop"代码去评估它，写得越多越觉得自己作为质量工程师的尊严正在被磨光。还有人吐槽同事完全把脑子交给 LLM：开会时说不出半句话，会后却在 GitHub 上贴出三页的"我后来想了想"长文，每一句都能看出是哪条 LLM 上文里抓出来的。

**第二种是"项目空转"**。一位老程序员说：去年一整年的项目今年一月被砍，被调到新组之后没什么实事可干，长时间空转、范围反复变更、什么都没上线，自我怀疑越积越深。"如果真被裁了，至少有时间好好准备面试。"

**第三种是"经济与年龄考量"**。一位用户回忆父亲当年被买断退休时获得的 package：加 10 年工龄计入退休金，外加医保到 65 岁——"我爸再没回头看过。可惜没人愿意花钱让我走"。另一位是已经申请 SSDI（社保残疾保险）半年但还没批的求职者，他想从"求职这件事"里被解雇。

**第四种是"管理之痛"**。一位 6 年大厂经验的中层管理者描述了"温水煮青蛙"——一位他原来非常尊敬的老同事变成他的直属经理，6 个月后矛盾彻底爆发，最后唯一的解法是他自己跨组跳走。这条评论是整个帖子里最长、最沉重的一条。

整个帖子的氛围其实并不真的"想被裁"——大多数人是在公开表达"我对当前状态非常不满，想换一种活法但不知道怎么开始"。和"Who wants to be hired"对照阅读，这是同一群人的另一面：白天体面发简历，深夜匿名喊话"求被裁"。

## HN 评论精华

1. **saltyoldman**：直接给整条线程打了个反问句——"Quiet quitting / firing 这些概念在 2024 年就死透了，兄弟。" 一个时代的劳工话术正在快速过期。

2. **HotGarbage**：一句话冷笑话："还在热切地等 AI 把我的工作抢走呢。" 同时也是真心话——很多人把 AI 替代当成了一种被动的退出机制。

3. **Ancalagon**：吐槽现状最精确的一条——"我现在的工作就是评估非确定性的 agent 流程，用更多 slop 代码去评估它，这正在迅速消磨我作为质量工程师的全部热情。" AI 工程化让"质量"这件事的定义本身在崩塌。

4. **jakedata**：上一代的体面退场已经不存在了。"我爸到我这个年纪时被给了买断 package：10 年工龄加进退休金、医保到 Medicare 启用，他再没回头。可惜没人愿意付钱让我走。"

5. **loudermachine**：典型的"项目空转之痛"——去年一整年做的项目一月被砍，新组也没真正的工作可做。"我在自我怀疑里挖坑。要是被裁了，至少能腾出时间好好面试。"

6. **locusofself**：本期最长也最具体的一条。大厂中层 6 年经验，一位曾经的资深 IC 兼 mentor 6 个月前变成他的经理，"温水煮青蛙"式恶化到上周彻底爆发，最后唯一的解法是他跨组跳走。"我不想被裁，但确实需要在新角色开始前先休几天。"

7. **slau**：另一位同事正在"关掉脑子靠 LLM 工作"——线下沟通时根本说不清楚，60 秒后却能在 GitHub 上贴出三页 LLM 论文，自己跟 LLM 在 PR 里互相辩论。"我能一眼看出哪段是 LLM 抓的旁支信息，把我们线下已经决定不做的东西又拉回来了。" AI 让协作变得更累而不是更轻。

8. **jwrallie**：不是工作量的问题，是"上下文切换"的问题。同时被太多项目分心，已经主动找老板要求退出大部分。给自己一个月的观察期——压力如果还不下降就走人。
