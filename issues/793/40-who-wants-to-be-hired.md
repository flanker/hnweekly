## 文章摘要

与同期"Who is hiring"对应的"Who wants to be hired"帖呈现了 2026 年 5 月求职端的全景。和招聘端的"AI 浓度变高"相对应，求职端最显著的两点变化是：**地理彻底全球化**与**资历分布的两极化**。

**全球化**：发帖求职者来自门洛帕克（CA）、温哥华、慕尼黑/牛津、维尔纽斯（立陶宛）、马恩岛、葡萄牙、印度诺伊达、匹兹堡——几乎每个时区都有人。绝大多数（接近 90%）明确接受或偏好远程工作，许多国际候选人主动写明"愿意通过 Deel/EOR 用本地货币结算"——劳动力市场已经在制度上分裂为"美元通道"和"本地货币通道"两条。

**技术栈**：后端依然是 TypeScript / Python / Go / Rust 四强，PostgreSQL 与 Redis 高频出现，Kubernetes / Terraform 是 DevOps 标配。前端 React / Next.js + TS 几乎一统天下，移动端 React Native / Flutter / Swift / Kotlin 各有其人。**AI/ML 相关技能（LLM 集成、RAG、agent 系统）出现在大约 30% 的帖子里**——不再是少数人的特长，而是几乎每个高级开发者都会顺手写在简历上的关键词。

**资历层**：资深工程师（10 年以上）越来越多地把自己定位为"founding engineer"或 fractional CTO，而不是寻找传统的全职岗位。同时不少人提到自己刚从 Cloudflare、Cisco Meraki 等大公司被 layoff——侧面印证了 2025–2026 年大厂仍在持续裁员。资历两极化的另一端是 AI-native 的初级开发者，他们用工具放大产出，而非靠经验积累。

**专精溢价**：金融支付系统、blockchain 基础设施、医疗合规这种"窄而深"的方向比"通用全栈"更容易在帖子里被注意到。这与招聘端"AI + 行业知识"的需求形成镜像。

2. **valzevul（温哥华）**：12+ 年的资深 iOS 工程师/咨询师，技术栈 Swift/SwiftUI/UIKit/Swift Concurrency/TCA/Core ML/Vision/Metal/OWASP MASVS。代表了"被裁后转向独立咨询"的资深路径——直接在自己的域名挂咨询页面。

3. **DavidOrGoliath（牛津/慕尼黑，前 AWS）**：寻找 MLE / Applied Science 岗位。LMU、牛津背景，Computer Vision/Robotics/Inference Optimization/AutoML/Forecasting——典型的"博士+大厂"履历，市场上仍然抢手但他选择 in-person preferred。

4. **pypt（维尔纽斯）**：明确写"美国东部时间重叠 + 与太平洋时间异步多年"——东欧候选人非常清楚自己的差异化优势就是"会异步、性价比高"。技术栈是后端 Python/TS/PostgreSQL/Temporal + AI agent 基础设施。

5. **david_allison（马恩岛）**：Kotlin/Android/TS/React/.NET/Java/CI/CD/可访问性/i18n。强调"愿意为合适的角色搬家、愿意 onsite onboarding"——这种灵活姿态在远程市场拥挤之后开始变成一种差异化卖点。

6. **praveen4463（印度诺伊达）**：愿意按美国时间工作但不愿迁居。技术栈非常宽：Python/Go/Node/TS/Java/React/Next/GCP/K8s/Terraform/Temporal。是印度高级开发者通过 EOR 进美国市场的典型路径。

7. **cjbconnor（匹兹堡）**：6 年经验的后端/平台工程师，远程 + 偶尔出差。从 DevOps 咨询起家，标配 Java/Spring/Python/Rust/K8s/Terraform/RabbitMQ/PostgreSQL/AWS/Prometheus。代表了"美国本土中等资历远程候选人"的标准画像。

8. **整体观察**：这份帖子的"长尾形态"比一年前更明显——大量国际候选人在抢有限的远程岗位，而本土资深工程师反过来在向"founding engineer / 咨询"漂移。Junior 与 Senior 之间的"Mid-Senior 中段市场"反而是最拥挤的红海。
