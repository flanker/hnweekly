## 文章摘要

《韦诺之战》（The Battle for Wesnoth）是一款开源的回合制策略游戏，以高幻想（High Fantasy）为主题，支持单人战役和在线/热座多人对战模式。这款游戏自 2003 年首次发布以来，已经发展了超过二十年，成为开源游戏领域最成功的项目之一。

游戏设定在一个名为韦诺的奇幻世界中，玩家可以参与多种冒险：夺回被篡夺的王位、逃离巫妖领主的追杀穿越大海寻找新家园、深入地底最深处锻造火焰宝石、保卫王国免受邪恶巫师亡灵大军的侵袭，或者率领一群幸存者穿越炽热沙漠对抗未知的邪恶力量。

游戏拥有丰富的特性：手工绘制的像素风格动画单位，配以半写实风格的对话肖像；17 个单人战役和 55 张多人地图可供选择；七大主要阵营中有超过 200 种单位类型，各具独特的能力、武器和法术；支持通过互联网、私有/局域网或热座进行多人对战；已翻译成超过 30 种语言；拥有高度可修改的引擎，结合了 WML 和 Lua 脚本语言；官方插件服务器上有大量玩家制作的内容，包括新战役、新阵营和新多人地图；跨平台兼容 Windows、macOS 和 GNU/Linux。

游戏可以通过官方网站下载，也可以在 Steam 和 itch.io 上获取。该项目的源代码托管在 GitHub 上，是一个真正意义上的社区驱动项目，长期以来依靠志愿者开发者和艺术家的贡献不断进化和完善。作为开源游戏的标杆项目，《韦诺之战》证明了社区协作可以创造出与商业游戏相媲美的高品质作品。

## HN 评论精华

1. **jjmarr** 强烈推荐韦诺之战的核心开发者之一 Charles Dang（Vultraz），他自 2012 年起就参与韦诺的开发，但 2024 年才大学毕业。尽管他是 GitHub 上最受欢迎的 C++ 开源项目之一的维护者，在当前残酷的应届毕业生就业市场中仍然很难找到工作。这条评论引发了人们对开源贡献在求职中价值的讨论。

2. **dhbradshaw** 分享了与孩子们一起玩韦诺之战长达约 20 年的经历。他指出这款游戏在 Linux、Mac 和 Windows 上都能运行，从未停止工作过。而与此同时，许多其他心爱的游戏已经消失或过时了。他认为优秀的游戏没有理由不能像国际象棋一样永恒，也许韦诺之战可以做到这一点。

3. **wiradikusuma** 询问是否有人能列出类似韦诺之战这样的高质量开源游戏清单——那种"我不敢相信这是免费的，即使收费我也愿意付款"的游戏。这个问题引发了大量讨论，人们推荐了许多优秀的开源游戏。

4. **rhdunn** 提出了对游戏的一个主要抱怨：治疗单位在治疗时不获得经验值。这意味着你需要将治疗单位投入战斗才能升级，而不是像设计初衷那样将其放在战斗单位后方。这与治疗单位初始血量低、非常脆弱的特点相矛盾，是一个长期存在的游戏设计争议点。

5. **rdevilla** 回忆了在大学时期用 Arch Linux Thinkpad 玩韦诺之战的美好时光。他最喜欢的功能是可以在地图上标记位置给队友看，他们会用角色名字命名这些标记点，记录一个英雄单位的冒险历程——"Ronant 的凯旋"、"Ronant 的复仇"、"Ronant 的愚行"，最终是"Ronant 的最后一战"。这展现了游戏如何创造出让人难忘的共同记忆。
