## 文章摘要

Framework 推出了其旗舰系列 Laptop 13 的"Pro"版本，将这条以可维修、可升级著称的产品线推向了一个新的工艺与配置高度。这一代最显著的变化是**机身材质的彻底升级**：从过去的镁铝合金混合结构改为整块 6000 系铝合金 CNC 一体加工成型，机身刚性、手感和质感都向苹果 MacBook Pro 看齐。Framework 在官网上也罕见地坦承借鉴了苹果的工业设计思路，这种透明态度反而获得了不少好评。

核心规格上，13 Pro 的看点也相当扎实：

- **处理器**：Intel Core Ultra Series 3（代号 Panther Lake）
- **内存**：最高 64GB LPCAMM2 LPDDR5X，**可升级**——这是 Framework 与 Intel 最新平台首批搭载 LPCAMM2 的笔记本之一，意味着内存不再是焊死的，可以将来更换或升级
- **存储**：最高 8TB PCIe Gen 5.0 NVMe
- **屏幕**：13.5 英寸 2880×1920 触控屏，3:2 比例，可变刷新率 30–120Hz
- **电池**：续航最长 20 小时（Netflix 流播放场景），容量比上代增加了 21%
- **键盘**：1.5mm 键程，多种语言布局可选
- **机身尺寸**：296.63 × 228.98 × 15.85 mm，重 1.4 kg
- **触控板**：搭载压电致动器的**触觉反馈**触控板（haptic touchpad），取代传统机械点按

依旧延续 Framework 招牌的**模块化扩展卡**——USB-C、USB-A、HDMI、DisplayPort、Ethernet、SD/microSD、3.5mm 音频等接口可自由组合；磁吸的彩色边框（Bezel）允许个性化外观；DIY 自组与预装两种购买方式同时提供。

特别值得一提的是 Framework 这次明确把 Linux 用户作为头等公民对待：Ubuntu 已认证，并支持 Fedora、NixOS 等主流发行版。在 Linux 笔记本生态依然相对小众的情况下，这种姿态对开发者社区有相当大的吸引力。

## HN 评论精华

- **向后兼容才是 Framework 真正的价值**：用户 chis 指出此次发布最让他印象深刻的并非新机本身，而是"这些升级几乎每一项都可以反向移植到旧款机型上"。这种对老用户的尊重在 PC 行业近乎绝迹。
- **没有兼容性 = 一台过价的笔记本**：jszymborski 接着补充道："正是因为 Framework 坚持给老用户提供升级路径，它才有了存在的理由。如果哪一天他们放弃了向后兼容，那它就只是一台过价的笔记本，比不过 ThinkPad 或者 Dell XPS。"
- **比台式机更可升级的笔记本**：lynndotpy 给出了一个相当戏剧化但真实的比喻——"Framework 在做一件了不起的事情：让笔记本拥有了比台式机更好的向后兼容和可组装性。台式机平台每六年就会换一个不兼容的插槽，而 Framework 五年都没让旧主板的老用户掉队。"
- **诚实的营销文案**：hgo 表扬 Framework 的发布会风格："他们公开承认借鉴了苹果，电池容量直接给 Wh 数字，没有满屏的市场学胡话。这在如今的科技产品发布里实在罕见。"
- **可升级的现实成本**：cogman10 提出了一个清醒的反例——"逐项升级一台五年的旧机器，加起来的成本可能跟买一台同价位竞品的新机差不多。所以可升级在技术上成立，但实际场景里能不能用得上是另一回事。"也就是说，向后兼容是一种**期权价值**，但要把它兑现仍需精打细算。
- **Linux 续航数据缺位**：多位用户提到 Framework 这次只发布了 Windows 下的续航数据，对一批等着上 Linux 的核心用户来说不够直观。Framework 的产品负责人 nrp 在评论区回应承诺会补上 Linux 下的具体测试。
- **价格未公布的隐忧**：评论区也有声音指出此次发布信息中价格相对低调，担心 CNC 全铝机身和高规格屏幕会把整机价格推到接近 MacBook Pro 的水平，反而稀释了 Framework 的"性价比 + 可维修"叙事。
