## 文章摘要

Reuters 这篇报道（原文返回 403 无法直接抓取，本摘要主要基于 HN 上引述的报道原文段落与上下文）披露：Meta 正在向其美国员工的工作电脑安装新的跟踪软件，用以捕获鼠标移动、点击和击键，并将数据用于其 AI 模型训练——这是 Meta 打造"能自主完成工作任务的 AI 智能体（agent）"的整体战略的一部分。该消息来自 Meta 内部备忘录，由 SuperIntelligence Labs 团队的一位 AI 研究科学家在专门频道发布。

按照报道：这套工具会在一份"工作相关应用与网站"白名单上运行，并会偶尔截图记录屏幕内容以补充上下文。值得注意的是 HN 评论者 loeg 指出的一个细节——所谓"工作相关应用与网站"包括 Gmail、Google Docs、甚至 Facebook 和 Instagram 本身，并没有为员工的个人账号留任何排除条款。Meta 发言人 Andy Stone 声称收集到的数据"不会用于绩效评估，也不会用于模型训练以外的任何用途"。

报道引述 Meta 自己的解释：训练目标是改善模型在"下拉菜单选择、键盘快捷键使用"等仍然薄弱的能力。HN 上多人指出这背后是更大的"agentic computer use"赛道竞争——OpenAI 通过 Operator、Anthropic 通过 Computer Use、xAI 据传也在开发"Human Emulators"，都在争夺"人类如何使用电脑"的真实轨迹数据。Meta 想用上自己数万员工每天产生的高质量、合法（毕竟是雇佣关系）的工作数据。还有评论者把它和 Meta 此前收购的 Limitless.ai、以及挖来 Scale AI 的 Alexandr Wang 联系起来——一条把"数据采集"作为传家手艺的链条正在成型。

法律和文化层面的争议非常激烈。报道未明确说欧盟员工是否覆盖，但 HN 上许多人指出在欧盟和英国，这种粒度的监控会触碰 GDPR、各国劳工委员会条款。即便在美国，员工对 Meta 承诺的"绝不用于绩效评估"几乎一致地不相信——刚经过几轮把员工标为"underperformer"再裁员的公司，再说"这次只是用来训练 AI"，可信度极低。

## HN 评论精华

- **dagmx**：抓住问题的本质——这与传统的"事后查日志"完全不同：这是"大规模主动监控"，意味着员工在工作时间内的任何分歧、抱怨、或与工作无关的私人沟通都不再有任何隐私期望，将极大压缩内部异议空间。

- **xvxvx**：尖锐地指出 Meta 发言人那句"数据不会用于绩效评估"是 horseshit。第一，员工被要求训练取代自己的 AI；第二，这份数据 100% 会影响绩效评估，毫无悬念。结合 Meta 此前用绩效审查作为隐性裁员工具的历史，这条评论赢得了广泛共鸣。

- **wrs**：用一句话点穿——"你期望全世界最了解隐私如何被滥用的 Meta 员工，去相信 Meta 这条承诺？" 

- **beloch**：极具画面感的类比——"这就好比你去毒品作坊上班，所有人都被要求脱光以确保没人偷'产品'。这是一个零信任环境，再加上随时被 AI 替代的恐怖。毒品作坊里的工人至少不会被解雇，因为雇主直接拿枪指着他们；Meta 员工则被工资和股票指着。"

- **lukeschlather**：从合规与安全角度提出关键问题——SRE/运维工程师的屏幕截图必然会包含密码、加密密钥、PII、客户数据等不允许任意存储的敏感信息。一旦这套训练数据集被攻破，将会构成史上最广义的内部访问风险，比 Equifax 那种身份信息泄露更严重；此外这种存储本身就可能违反 SOC2、PCI、GDPR 多项条款。

- **AgentOrange1234** 风格的安全担忧由 **lukeschlather** 等多人重复呼应：屏幕截图 + 击键流 = 公司最敏感数据的实时拷贝，"Meta 怎么打算保护这些数据？员工是否能临时关闭以避免在 cat 一个根 SSH 密钥时被记录？"

- **gip** / **general1465** / **nitwit005** 提出了相反方向的疑问：单纯从"训练 agent 模型"的有效性来看，光抓鼠标轨迹和击键并不会得到为什么这样做的因果信息。"agentic" 模型应该用 accessibility API 而不是模仿人类点鼠标，技术路线本身就值得怀疑。

- **vrganj**：直接呼吁组织化——"工程师们，你们被自由意志主义和个人崇拜麻醉太久了。资本家从来不是你的朋友。唯一被证明有效的劳工权利保障方式是集体行动。是时候组建真正的科技工人工会了。"这条评论代表了 HN 上越来越显化的"科技工人阶级觉醒"线索。

- **dbgrman** 引用 Andrew "Boz" Bosworth 的内部备忘录原话："我们如此深信连接人类，以至于任何能让我们连接更多人更频繁的事都是 de facto good。"——以此说明 Meta 这种激进监控并非偶然，而是企业文化深处"目的合理化手段"信条的延续。
