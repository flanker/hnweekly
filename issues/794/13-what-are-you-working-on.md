## 文章摘要

这是 HN 每个月例行的 **"What are you working on?"** 帖——一句最朴素的提问，钓出来的是几百个独立开发者、研究者、副业玩家、转型者在自己后院里点的小灯。和"Show HN"不同的是，这里贴的东西不一定能 demo，更多是"我现在脑子里有这个事"，所以会有比平常更多的**真硬件、真业务、真生活**项目混进来——本月这一期尤其明显。

可以读到的几个高频信号：**一、AI 周边的工具市场仍在长**——permission 层、coding agent、context 管理这种细分位置在不断有人下手；**二、独立游戏和每日小游戏继续在 HN 群体里活跃**——这次有人发布到 Steam，有人坚持做每日填词；**三、"非软件"项目肉眼可见在变多**——电子表时钟、潜艇模拟器、铜矿冶炼做戒指、纺线钩织、动物声学诊断——HN 群体在用代码以外的方式自我表达；**四、地域 / 主权类项目（only-eu.eu）和欧洲夜火车这类带"地缘 + 基础设施"色彩的副业**也在涌现。这一期的"working on"几乎是一张 2026 年独立创造者的全景小图。

## HN 评论精华

- **wwwkieran**：在做 **Drawers**（drawers.computer），macOS 上"每个项目独立 dock 和 space"的专注力工具——可以把 dock 里的资源细化到**单个 Slack 频道**而不是整个 Slack。免费下载找 beta 用户。

- **paulhebert**：日填词游戏 **Tiled Words** 满 6 个月了——他和老婆每天出一道新题坚持半年，被 Forbes 报道。下一步开放**用户提交题目**和出题工具。HN 上独立 dev "用爱发电做每日游戏"的代表案例。

- **druskacik**：欧洲夜火车搜索 **trainbot.eu**——和飞行机票引擎不一样，**不要求你输入具体日期**，而是输入出发地后**推荐"未来某天可以去的有意思的目的地"**。在补足跨多供应商的夜火车数据。

- **luigi_123**：第一次在 HN 发帖——他在做一个**专门给鼓写谱的文本 DSL**（灵感来自 ABC Notation）。同一份文本既能渲染 PDF 乐谱，**也能当 DrumMania 风格的节奏游戏**玩。狗食半年了，下一步是带导入歌曲的 IDE。

- **seydar**：原潜艇兵——做了个 iPhone app，**用手机声音诊断电力变压器健康**（绕组、铁芯、相位是否失衡）。他笑称声学诊断在电力领域"研究意外地少"，他用声纳那套思路从基础物理一点点逆推。Xcode 让他想砸电脑。

- **mstaoru**：练了 20 年举重的开发者——在做一个**基于新一代 IMU 的杠铃路径与速度跟踪传感器**，比摄像头方案精度更高、更便宜。原型阶段"Raspberry Pi 绑在杠铃上"。明确标注 [NO-AI]——他觉得这个交叉领域**LLM 还没污染**，难得清净。

- **shinrak**：终于发了第一款 Steam 游戏 **"balls"**——连锁反应弹球+combo。他描述开发过程是"一个小玩具，加感觉对的功能就保留"，"很难校准数字"。还在打补丁，考虑加 idle 模式。

- **madman_dev**：维护 **only-eu.eu**——一个**欧洲软件 / 产品替代品目录**（云存储、邮箱、VPN、浏览器、手机、自行车……），30+ 类目下约 175 个产品。下一波要加：personal health、商业邮件 / newsletter。**地缘和数字主权**意识在欧洲独立开发者群体明显抬头。

- **paulnovacovici**：白天写应用的他被 Claude Code 和 Codex 终于"哄"进了单片机——做了个**"全息"显示器，给任何海滩展示冲浪预报**，朋友帮他做了外壳和机械结构。HN 经典副业组合。

- **wpietri**：在芝加哥 Loop 区开了一家 **2900 平方英尺的非营利弹球博物馆**（theflip.museum），即将一周开七天。"第一次开实体零售业务，从'细节'里学了一大堆。"

- **markmnl**：**fmsg**——一个**像 email 那样按域名分布的开源即时通讯协议**。地址形如 `@user@domain`，谁都能跑节点，消息之间用密码学父哈希串联成线程。原本想给"人对人"做，现在因为 "人对 agent" 和 "agent 对 agent" 反而显得正当时。

- **SdtEE**：做了 **Hitomi**——一个**能 O(1) 时间打开任意大 CSV / 日志文件**的本地分析器。秘诀是增量组合解析器：把 stream 切块后送进流水线，**第一个小 chunk 处理完 UI 就出现**。Native，不带 Electron 污染。

- **AsmaraHolding**：**X/D Loom**（xdloom.com）——给改装车爱好者做接线图。起因是他给自己 Nissan D21 做了发动机换型 + 一堆改装，**一年后忘了线是怎么接的**。市面上工具都是企业级 $200/月。

- **inslee1**：自家配送业务用的 **toanoa.com 物流管理系统**已经撑过 10 万订单——做了智能订单 batching + 路线优化，可以在配送窗口内交错多个订单的子任务。后端响应时间在几十毫秒级。

- **niothiel**：**cardcast.gg**——用网络摄像头远程联机打 **万智牌（MTG）**。从经典计算机视觉一路换到 transformer 模型识别牌面，他朋友圈玩起来比 Spelltable 还顺手。

- **epaga**：二战潜艇模拟器 **Silent Shark**。免费 alpha 在跑，几个月内上 Steam / 商店。"最爽的时刻是 stadimeter（用桅杆高度 + 视角推距离）不靠任何作弊地就**算对了，因为数学就是这么干的**。"

- **jballanc**：发了第一篇预印本——**RVW**，一种**支持在线持续学习且不灾难性遗忘**（catastrophic forgetting）的 Transformer 改造。在 SmolTalk 上 PPL ~18，参数只有 4B。在扩到 Llama 3.2 风格 2B 版本。

- **gustavopezzi**：从大学全职教职辞职了，pikuma.com 也暂停更新——目前**专注复习数学物理，给儿子做 homeschool**，下半年要把"每周只上一次网"作为日常。HN 上这种"主动下线"的故事越来越常见。

- **schipperai**：**coding agent 的权限层**——他用 benchmark 证明 95% 的 Bash 调用可以**用确定性分类器**判定为安全或危险，不需要 LLM 介入。结论：用 LLM 做 permission review **像用整个数据中心来按一下开关**。

- **koeng**：**自己冶炼矿石做戒指**——金太硬、银有毒，他选孔雀石（malachite），开车去 Copperopolis 挖原矿，配 flux 和炭，熔出纯铜，浇成铸锭，敲成戒指。"用自己采的金属戴自己手上。"

- **needcaffeine**：**OnetimeFax.com**——按次付费发 / 收传真，不用订阅。最初版本是他疫情期间为了给法院发陪审员豁免请求自己写的脚本，后来产品化了。

- **maxbond**：在学**钩针**——把它当成一种"数学有趣的"手工活——基础针法是滑结的串联，**每一针都是可逆变换**；理论上你可以找到线头从最后一针开始反向解开整件作品。"如果犯了错总能退回去重试。"
