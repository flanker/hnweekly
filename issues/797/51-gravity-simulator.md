## 文章摘要

Gravity 是开发者 qunabu 制作的一个交互式太阳系模拟器（"Solar System Model"），项目开源在 GitHub 上。它在浏览器中以 3D 形式呈现各天体的运动，使用了来自 Solar System Scope 的真实地球贴图（基于 CC BY 4.0 授权），力求提供逼真的视觉效果。

模拟器带有引导式的分步教程，让用户从牛顿力学的视角逐步过渡到爱因斯坦的相对论引力观，并配合相机巡游来"游览"整个太阳系，兼具趣味性与科普价值。用户可以与场景交互、选择不同天体进行观察。

作者 qunabu 在讨论中积极回应了社区反馈，并发布了若干修复补丁来解决评论者指出的问题。

## HN 评论精华

**rfgplk**（置顶评论）肯定了模拟器的高效，但批评它把牛顿引力和相对论引力人为割裂开来，认为这会造成困惑——因为爱因斯坦引力才是更普遍的一般情形。他建议借鉴 Alessandro Roussel 的时空可视化方式来提升教学效果。

**jumploops** 称赞模拟器准确地呈现了太阳系运动的 3D 螺旋（helix）轨迹，而不是那个曾在网上流传的错误"漩涡（vortex）"说法，并建议未来可以扩展到银河系尺度。

**SamBam** 就教程的物理表述展开讨论，指出教程暗示"存在唯一一个稳定的轨道速度"是不准确的——调整轨道速度只会改变轨道大小，而非简单地决定是否逃逸。

**nonethewiser** 由轨道稳定性的"小概率"引发感慨，讨论了什么因素可能扰动地球轨道，并联系到了人择原理。

**twosky** 喜欢这种沉浸式的巡游体验，但希望相机能更自由，并报告了一个 bug：选中火星时有时会显示成另一颗行星。

**kmaitreys** 推荐了 rebound 这个库，它被专业的行星形成研究用于严谨的 N 体模拟。
