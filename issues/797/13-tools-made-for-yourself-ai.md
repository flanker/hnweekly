## 文章摘要

发问者邀请社区成员分享：自 AI 技术（尤其是大语言模型）兴起以来，他们为自己亲手打造了哪些工具和小程序。

这个话题切中了 AI 浪潮下一个有趣的现象——编程门槛的降低让越来越多人能够把脑海中的奇思妙想快速变成可用的产品。回复中既有音频实验类的创意小站，也有提升日常效率的实用工具，还有非开发者借助 AI 完成的完整商业项目。

这些分享生动展现了 AI 如何赋能个人创造力，让「为自己挠痒痒」式的工具制作变得前所未有的容易。

## HN 评论精华

**yungbeto**：分享了一系列音频实验项目，包括 Noisetable.xyz（一组以 VCR 录像机界面为灵感、基于随机性的音频「频道」集合）、Concrete95.net（做成 Windows 95 风格的具体音乐 web 应用）以及 Windso.me（一个不让你自己选择采样的采样步进音序器）。

**michaelbuckbee**：重点介绍两个项目：Moniker（一款免费工具，利用本地 AI 根据文件内容智能重命名文件）；以及 HutchDB（一个 MCP 服务，让你能对 AI 说「把这个存到 Hutch」，自动处理 schema 创建、看板/时间线/网格/日历等多种 web 视图以及分享功能）。

**netcoyote**：列举了多款工具，包括 Sandvault（在独立的 macOS 用户账户中运行 agent，并用 sandbox-exec 加固）、Push10K（一款追踪自己「一万个俯卧撑」进度的 iOS App），以及用 Zola 静态站点、Sveltia CMS、Cloudflare Pages/Workers 搭建的全 AI 编写博客。

**alphaBetaGamma**：作为非开发者，打造了一门受计算机科学启发的珠宝生意——用自定义 DSL 设计珠宝，由 LLM 生成 CAD 解释器，并用 Claude Code 设计店面。产品包括能计算葛立恒数的 lambda 图（Tromp 图）。

**asciimoo**：开发了 Hister——一个面向网站和本地文件的全文索引器，会自动保存浏览器渲染过的所有访问过的页面，提供注重隐私的离线搜索，特别适合「我记得看过但想不起在哪」这类回忆型检索。

**melvinroest**：做了一个语音备忘录 App，支持意识流式录音，再由 Claude 把转写文本整理成有条理的笔记，让他「只需边想边说，就能得到一个帮我梳理思路的东西」。
