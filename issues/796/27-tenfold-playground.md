## 文章摘要

Tenfold 是研究机构 Ink & Switch 开发的一个基于网页的创意工具，让用户通过过程式（procedural）代码来设计自定义字形（letterform）。它把视觉设计与"社会化设计"原则结合在一起，营造出一个鼓励创意表达的有趣环境。该项目源自 Todd（视觉设计）与 Ivy Reese 的合作，是 2025 年底 Automerge 官网改版工作的延续。

项目探讨的核心问题是：如何让非程序员也能轻松上手过程式艺术，同时保持美学上的统一；更宏观地说，它在研究"刻意设置的约束与摩擦"如何反而能激发而非限制创造力。

几个关键理念：

**约束即解放**。这个 API 故意把用户限制在固定粗细的白色线条和圆弧上。正如 Ivy 所说，约束能"把你从面对空白页时的种种犹豫中解放出来"，让创作空间显得平易近人，而不是无边无际、令人却步。

**通过"假墙"实现社会化设计**。团队没有强制封死所有限制，而是故意留了一些"逃生口"（如可以访问 window 对象）。这造就了一些"假墙"——用户发现自己竟能引入状态、做游戏、用 WebGL，于是产生一种"我聪明地打破了规则"的成就感，同时也拓展了创作的可能性。

**归一化的数值空间**。系统用 0 到 1（归一化）和 -1 到 1（裁剪）作为坐标与参数范围，带来直观的数学性质——例如相乘仍能保持在范围内。

在实现上，用户在一个实时编辑器里编写 JavaScript，代码每一帧都会重新求值来绘制动画，由 `time`（在 0 到 1 间往复的锯齿波）、`x` 和 `y`（光标位置）等变量驱动。API 提供了符合人体工学的辅助函数，如以"圈数（turns）"为单位的 `cosn()`、`sinn()`，自动起笔以及便捷的范围映射函数。代码通过 Automerge 立即同步到 service worker 和其他用户，体现了 Ink & Switch 在**可塑软件（Malleable Software）**与**可编程墨水（Programmable Ink）**方向的研究主旨。
