## 文章摘要

Muxcard 是开发者 krauseler 打造的一台功能完整、却被塞进了真正信用卡尺寸（厚度约 1mm）的微型电脑。正如作者所言："一台真正只有信用卡大小的、完全可用的电脑。"这个项目最令人惊叹的地方，正是它在如此极端的空间限制下依然保持了完整的计算能力。

在硬件配置上，Muxcard 采用 ESP32-C3 处理器（也有 nRF52 的备选方案）；显示部分是一块 1.54 英寸的柔性电子墨水屏，分辨率 200×200 像素；连接能力方面集成了 NFC（基于 RC522 芯片）并支持 WiFi；还配有一颗 LIS2DW12 加速度计用于感知运动；供电则来自一块超薄的 30mAh 锂聚合物电池（作者目前正在测试容量更大的版本）。所有电子元件都被完整地封装在卡片内部，没有任何外部组件。

为了攻克极端的空间约束，这个项目在工艺上做了大量创新：作者用 3D 打印机充当光刻工具，手工蚀刻柔性 PCB；将元件巧妙地布置在一个个"岛屿"上以分散机械应力；由于没有任何标准连接器能塞进这么薄的空间，电子墨水屏的排线甚至是逐根手工焊接的。目前这台概念验证原型机已经能成功运行。作者设想的应用场景包括极简数字钱包、渗透测试工具、智能家居仪表盘、离线凭证存储，以及一张带功能屏幕、令人过目难忘的电子名片。下一步，作者将专注于耐久性测试、电池优化和可量产性等挑战，再考虑商业化。

## HN 评论精华

1. **开发者亲自现身**：开发者 krauseler 本人在评论区露面，幽默地表示这大概是他收到的第 100 个"它能跑 DOOM 吗？"的提问，如今这甚至已经成为 GitHub 上的一个正式 issue。他打趣道："那是不是意味着我现在真得把它实现出来了？"——呼应了 HN 经典的"Can it run DOOM?"梗（ohlookcake 等多人都在追问）。

2. **对"厚度"的意外认可**：用户 mabster 坦言自己点进页面时本想吐槽"这根本不是信用卡尺寸，因为太厚了"，结果难得地被惊喜到了——这次它是真的做到了信用卡的厚度，他对作者大加赞赏。

3. **"它到底算不算电脑"的争论**：用户 lxgr 玩了个技术梗——"一台真正信用卡大小的、完全可用的电脑？吹毛求疵一下：带芯片的信用卡本身就已经是一台完全可用的电脑了。"另一位用户 goodpoint 则更直接地断言"它不是一台电脑"，引发了关于"什么才算电脑"的小范围讨论。

4. **银行与支付场景的想象**：用户 mmmehulll 很喜欢这个项目，并畅想能在卡上查看和执行各类银行交易，"就像硬件钱包 Ledger，但所有功能合而为一"，甚至幻想实现无需刷卡机的"卡对卡转账"。

5. **历史与反向脑洞**：用户 asdefghyk 由信用卡尺寸联想到 2000 年代初流行的"Linux 名片光盘"（Bootable business card），那是一种信用卡大小的可启动光盘，能装下 50MB 到 100MB 的精简 Linux 系统。而 cbdevidal 则开了个反向脑洞——他希望有人反其道而行之，做一台全尺寸的笔记本电脑，里面用 ESP32 跑 tiny386 和 Windows 95。此外 rbanffy 希望屏幕能占据更大面积（哪怕牺牲一点厚度），iberator 则建议在上面跑只需约 80KB 内存的 16 位 Unix v6。
