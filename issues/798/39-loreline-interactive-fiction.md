## 文章摘要

Loreline 是一套开源的语言与工具集，专为编写互动小说、游戏对话和分支叙事而设计。它由 Jérémy Faivre 创作，以 MIT 协议发布，在面向写作者保持易用的同时，也支持复杂精巧的叙事需求。

工具集的核心是 **Loreline Writer**——一个免费应用，让作者用 Loreline 脚本语言来撰写故事。此外还提供浏览器内的 Playground 供用户免安装试验，以及面向开发者的完整文档。Loreline 在设计上兼顾简洁与深度：既能让「写故事和对话变得简单」，又提供「高级分支、状态管理、函数」等能力来驾驭复杂剧情。

在工程集成方面，Loreline 可对接主流游戏引擎和编程环境，包括 Godot、Unity（C#）、各类 JavaScript 框架，以及 Python、Lua、Haxe、Java 和 C++，确保跨开发环境的可移植性。它还内置本地化支持，兼容 PO、XLIFF 等行业标准格式，无需专有工具即可接入专业译者的工作流程。项目托管在 GitHub 上，源码、问题追踪和社区贡献完全开放，既适合个人创作者，也适合较大的开发团队。
